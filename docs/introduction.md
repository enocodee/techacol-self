# Techacol-self
## Description
`Techacol-self` is an **programming**, **adventure** and **role-playing** game developed in [Enogine](https://github.com/enocodee/enogine).

## Gameplay
### Coding
- Each equipment has its own terminal for some special abilities (more speed, ...) that is able to access by another entities (includes players, entities ingame). [core]
- Each entity has a terminal that is allowed to access and disable abilities of them by players. (recovery after 5-10s) [core]
- TODO: Do all terminals should use the same database for packages?
- NOTE that all commands in terminals are pre-defined, allowing write, read, read & write based on their usage.
